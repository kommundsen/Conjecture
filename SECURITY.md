# Security Policy

## Supported Versions

Conjecture is pre-1.0. No version currently receives long-term security support.

| Version | Supported |
|---------|-----------|
| < 1.0   | Best effort only |

## Reporting a Vulnerability

Please **do not** open a public issue for security vulnerabilities.

Report privately via [GitHub Security Advisories](https://github.com/kommundsen/Conjecture/security/advisories/new). You will receive a response within 7 days. If confirmed, a fix will be released as soon as practical and the advisory published after patching.
