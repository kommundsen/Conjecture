# Conjecture.Protobuf

Protobuf-driven generation for [Conjecture](https://github.com/kommundsen/Conjecture) property-based testing. Produces `JsonElement` values shaped by a `MessageDescriptor` (or a `T : IMessage<T>` type), useful for fuzzing gRPC services, queue consumers, or any code that decodes a protobuf-shaped payload.

## Install

```
dotnet add package Conjecture.Core
dotnet add package Conjecture.Protobuf
```

## Usage

```csharp
using System.Text.Json;
using Conjecture.Core;
using Conjecture.Protobuf;
using Google.Protobuf;
using Google.Protobuf.Reflection;

// Generic form: descriptor inferred from the C# type generated by protoc
Strategy<JsonElement> orders = Strategy.FromProtobuf<OrderProto>(maxDepth: 5);

// Or pass a descriptor directly (e.g. when the type is loaded dynamically)
MessageDescriptor descriptor = OrderProto.Descriptor;
Strategy<JsonElement> sameAsAbove = Strategy.FromProtobuf(descriptor, maxDepth: 5);

JsonElement sample = orders.Sample();
Console.WriteLine(sample.ToString());
```

`maxDepth` bounds recursion through nested message and `repeated` fields so generation always terminates.

## Types

| Type | Role |
|---|---|
| `Strategy.FromProtobuf<T>(maxDepth)` | Strategy from a `T : IMessage<T>` whose `Descriptor` is reflected. |
| `Strategy.FromProtobuf(descriptor, maxDepth)` | Strategy from an explicit `MessageDescriptor`. |
| `ProtobufFieldStrategy` | Lower-level building block for per-field generation. |
