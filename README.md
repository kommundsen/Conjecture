# Conjecture.NET

## Concept

Create a .NET port of [Hypothesis] that feels like it was built for.NET.

## Credit to the HypothesisWorks/Hypothesis Team

This project is a .NET port of [Hypothesis] for Python. The concept of this project builds on and would not be possible without the work of [David R. MacIver](https://www.drmaciver.com/) and [Zac Hatfield-Dodds](https://zhd.dev/), as well as the many other [authors](https://github.com/HypothesisWorks/hypothesis/blob/master/AUTHORS.rst) of that project.

[Hypothesis]: https://github.com/HypothesisWorks/hypothesis

## Installation

```
dotnet packages add Conjecture.NET
```
